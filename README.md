# Tech BuilderZ Unified Suite

This repository combines three projects used for the education-focused digital services business:

1. `marketing-site` - Tech-BuilderZ sales and services website
2. `template-engine` - reusable school website template generator and SEO assets
3. `vidyalaya-dbms` - Vidyalaya DBMS platform (backend + frontend)

## Repository Structure

- `marketing-site/`
- `template-engine/`
- `vidyalaya-dbms/`

## Quick Start

### 1) Marketing Site

Serve locally with any static server from `marketing-site`.

### 2) Template Engine

From `template-engine`:

```bash
npm install
npm run build:all
```

### 3) Vidyalaya DBMS

Backend and frontend setup live under `vidyalaya-dbms`.
Refer to `vidyalaya-dbms/README.md` for environment setup and run commands.
